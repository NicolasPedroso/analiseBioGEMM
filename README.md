## ⚙️ Pré-requisitos
- Python 3.11.2
- gcc 12.2.0

## 🔧 Instalação
Clone o repositório e instale as dependências:
```bash
git clone https://github.com/NicolasPedroso/analiseBioGEMM
```
## 💡 Como Usar
```bash
cd analiseBioGEMM

gcc analises.c -o analises

./analises <pasta-raiz>

python3 graficos.py tabela
```
